Special Matrices

Jacobian Matrix

The (i,j)-th element of the Jacobian matrix is defined as the derivative of i-th function with respect to the j-th indeterminate.

  • jacobian(f, indets) -- where f (polynomials) and indets (indeterminates) are vectors of RingElem, all belonging to the same PolyRing. Throws if both f and indets are empty (cannot determine the ring for constructing the 0x0 matrix).
