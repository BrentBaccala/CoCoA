Maintainer documentation for the class GPoly

Also this part should be written by Massimo Caboara, but, as I am the author and maintainer of the reduction code, I write some notes here.

A GPoly contains some member fields which often depend solely on the field myPolyValue: myLen, myWDeg, myLPPwMask, myComponent After a reduction we change the value of myPolyValue and the above fields can be updated accordingly calling: myUpdateLenLPPDegComp(); NB: if myPolyValue is 0 the fields myWDeg, myLPPwMask, myComponent are unreliable (intrinsicly undefined).
